# MedStopLoss

Marketing website for Medical Stop Loss insurance — educating employers and brokers on self-funded health plan protection.

## Pages

- **Home** — Value proposition and hero
- **About** — Company overview
- **Services** — Stop loss product details
- **Brokers** — Broker resources and partnerships
- **TPAs** — TPA integration info
- **Resources** — Educational content and guides
- **Contact** — Full contact form

## Stack

- React + Vite
- Deployed on AWS (via GitHub)

## Run

```bash
npm install
npm run dev     # http://localhost:5173
npm run build   # Production build → dist/
```
